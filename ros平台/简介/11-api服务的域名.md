# 1.1 API服务的域名URL

---

# 服务地址

```
http://ros.ai
```

# 参数说明

| Field | Type | Description |
| --- | --- | --- |
|  | string | 用户当前位置，Location实体的一个实例:`location:{"latitude": "31", "longitude": "121"}`,分别传入经度和纬度数值。 |

# Code范例

```
{ 
"reference_time":1447499684824,
 "timezone":"Asia/Shanghai", 
"location":
        { "latitude":"31.215",
         "longitude":"121.609" }
} 
```
